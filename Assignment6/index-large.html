About this Project

From my testing of the magnifier, I found there are three different views: full screen, lens, or docked. Each of them could use different situation. For example, when you want input something, full screen should be better. when you read something, lens should be better. For the docked viewAnd I think it almost unuseful, we can remove this view. I want to add a design feature that the users can change lens size. Users can choose the most comfortable size.
